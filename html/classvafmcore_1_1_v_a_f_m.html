  argkw

Add a circuit of type 'ctype' named 'name' to the setup.

This looks into all the loaded modules whose name starts with 'vafmcircuits' and finds the first class named 'ctype'. It then instantiate the class.

Parameters:
ctypeName of the class of the circuit to add.
nameName of the circuit in the setup.
**argkwKeyworded arguments for circuit initialisation.
Returns:
Reference to the created circuit.
