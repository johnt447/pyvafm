Detailed Description

Oscillator circuit.

Creates sine and cosine waves with the specifics given by the inputs.

  • Initialisation parameters:
    • pushed = True|False push the output buffer immediately if True
  • Input channels:
    • amplitude
    • frequency
    • offset value
  • Output channels:
    • sine wave
    • cosine wave
